# Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

---

## 🚀 Features

- Two-player gameplay
- Winner detection
- Game reset functionality
- New game option
- Responsive design
- Clean UI with modern styling

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript

## 🎮 How to Play

- Player X starts first
- Click on any empty box to place your move
- First player to align:
  - 3 rows
  - 3 columns
  - or 3 diagonals
  wins the game
- Click **Reset Game** to restart
- Click **New Game** after a winner is announced

## 🧠 Game Logic

The JavaScript handles:

- Player turns (`X` and `O`)

- Resetting the board
